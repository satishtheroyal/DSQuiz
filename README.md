## What is DSQuiz?

DSQuiz is a full-stack quiz and assignment management platform built specifically for engineering college faculty. Instead of manually creating question papers every exam, faculty can generate topic-wise questions using AI in seconds.

Built by: **Satya Meda** — Assistant Professor & HOD, BVC College of Engineering

---

## Features

- **AI Quiz Generation** — Generate topic-wise questions instantly using Anthropic Claude API
- **Live Quiz Sessions** — Conduct real-time quiz sessions using Socket.io
- **Role-Based Access** — Separate dashboards for Faculty and Students
- **JWT Authentication** — Secure login system
- **PDF Export** — Download question papers with one click
- **Result Analytics** — Student performance tracking
